Wheelchair basketball application

Abstract

The concept behind our project was to produce an activity tracker for wheelchair basketball players to view their key statistics and monitor progression. Our solution is a mobile application that receives data from sensors before analysing it to produce stats. The application displays the results on a user interface, which the user can navigate through to find information from previous matches and averages over set periods of time. So far, we have developed the user interface and created a json file generator that produces mock data, much like that from the Fitbit API.

Key Features

Our application offers a number of key features including the following:

• Unique log in account for each user

• Option to log in with facebook

• Display of data including heart rate, top speed, average speed, calories burnt

• Data stored in the cloud
