TAWA: Tools for

Assistive Wellbeing Analysis

The concept behind our project is a set of tools that can be used for further research into the analysis of wellbeing for assistive technologies. The aim was that each element of our project can be extracted and utilised for further development processes. To provide this, we have developed aspects that cover the visualisation of data, retrieval of health-related data from a Fitbit API, a data-emulation generator linked to the cloud and a application that displays data on a user interface for an end user.

https://www.youtube.com/watch?v=ZgbTAxwKl8M&feature=youtu.be

Key Features

Our application offers a number of key features including the following:

• Json file generator that emulates data from a Fitbit API

• Android mobile application for end user, integrated with Json file generator

• Data visualiser for data from ARCCS sensors

• Application to recieve data from a Fitbit API

• Storage of data to the cloud
