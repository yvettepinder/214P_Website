User Manual

GameOn! is a mobile application to view analysis of a wheelchair basketball training session. The application allows you to view your statistics for each training session in addition to your average results over a number of games. This data is displayed on a graph for easier comparison. The following manual will run you through how to use the application.

Logging in

When the application is initially launched, the following welcome page will be displayed. In order to log in, select the option to continue with Facebook. This will take you to a page where you should enter your account details. After entering your details, press continue and you will be redirected back to the app and will now be signed in.

Adding Training Sessions

After logging in, you will be taken to the main dashboard. The first time you use the application, this will obviously be empty as you will not have entered any sessions yet. In order to add a training session, select the 'View Training Session' button from the main dashboard, then press the circular button on the bottom right hand corner. This will take you to the screen where you can enter the details of the training session. Pressing 'Create Session' will complete the process, and the session will be added to your list of training sessions.

After adding a training session, the main dashboard will be updated and contain information about the training session. The graph displays the data for a specific statistic for each session. As you add more sessions, you will see these added the graph, colour coded according to the session.

To change the statistic being displayed, horizontally scroll through the icons and press on one to select it.

Navigating

To navigate through the application, press the icon in the top left corner. This will open the navigation bar where you can select what page you would like to visit.

Deployment Manual

We used Android Studio for the entire process of development so that in the end the project could be easily installed on any phone running one of the latest versions of Android. This being said, once one has the source code and Android Studio installed on a computer, these are the following steps to have everything up and running:

Prerequisites

  • Android Studio
  • Source Code
  • Android SDK
  • Android Device

    • Step 1. Open Android Studio. The following window should appear:

    Step 2. Click on "Open an existing Android Studio Project". From there navigate to the directory of the source code and click on it, then open.

    Step 3. Wait a couple of seconds for Android Studio to import all the necessary modules. After that, click on the green triangle pointing to the right (the Run button). This will build and run the application on an emulator or a connect device.

    Step 4. Choose the device you want to deploy the application on.

    Step 5. You are all set. Enjoy using your application.

    (Additional) Step. We alternatively have an APK file for our application which can be downloaded directly from an android device to run the application instantly without Android Studio.
